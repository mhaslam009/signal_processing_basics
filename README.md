# signal_processing_basics
This is an effort in an engineering world to basically teach anyone with basics of signal processing for which i have remained curious and yet to build anything yet. 
### Relationship Between Level 1 and Level 2

| Level 2 Engineering Operation | Built Mainly From Level 1 Operations |
| :--- | :--- |
| **Filtering** | Addition + Multiplication + Delay |
| **Amplification** | Multiplication |
| **Attenuation** | Multiplication |
| **Modulation** | Multiplication |
| **Demodulation** | Multiplication + Filtering |
| **Convolution** | Addition + Multiplication + Delay |
| **Correlation** | Multiplication + Addition + Time Shift |
| **Sampling** | Time indexing |
| **Quantization** | Division (step size), rounding, multiplication |
| **Transformation (DFT/FFT)** | Addition + Multiplication (complex exponentials) |

---

### Recommended Progressive Notebook Structure

* **Level 1: Basic Mathematical Operations** (Foundational signal manipulations: addition, multiplication, time shifting, time scaling)
* **Level 2: Engineering Operations** (Core DSP building blocks: filtering, amplification, modulation, convolution)
* **Level 3: Signal Properties** (Quantitative metrics: energy, power, frequency, phase, bandwidth, SNR)
* **Level 4: DSP Systems** (System-level behavior: LTI properties, FIR/IIR filters, impulse response)
* **Level 5: Frequency Domain** (Spectral analysis: Fourier Transform, Discrete Fourier Transform, FFT, STFT)
* **Level 6: Real-World DSP Applications** (End-to-end pipelines: speech enhancement, audio/image processing, wireless communication, biomedical signals)
